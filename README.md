# Smart Lighting Project

## Project Overview
This project is designed to control lighting in specific areas of a home using a smart system. It turns on the porch light for 30 seconds and the hallway light for 60 seconds when a push button is pressed.

## Features
- Automatic control of porch and hallway lights based on button press.
- Configurable time for each light.
- Simple and intuitive user interface.
- Low power consumption.

## Hardware Requirements
- Arduino or similar microcontroller
- Push button
- Porch light (compatible with relay control)
- Hallway light (compatible with relay control)
- Relay module
- Power supply
- Jumper wires and breadboard for prototyping

## Setup Instructions
1. **Connect the Push Button**:  Connect the push button to the microcontroller according to the pin configuration in your code.
2. **Setup Relay Modules**: Connect the porch and hallway lights to the relay module and then to the microcontroller.
3. **Upload the Code**: Use the Arduino IDE to upload the code to your microcontroller.
4. **Power the System**: Ensure your system is powered and the lights are connected properly.

## Usage Details
- Press the push button to activate the lights.
- The porch light will turn on for 30 seconds.
- The hallway light will turn on for 60 seconds after the button press.
- Make sure all components are properly connected for the system to function as intended.

## Conclusion
This smart lighting system enhances home automation by providing a simple solution for controlling lights effectively while ensuring convenience and energy efficiency.
